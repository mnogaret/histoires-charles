# histoires-charles

Projet personnel et utilitaire d'édition d'un livre des mémoires de mon grand-père.

Le Makefile génère les PDF à partir des md.

Un script permet d'envoyer le texte brut à Google pour mise en forme sommaire en md (il faut d'abord définir la clé d'API dans le script).

source venv/bin/activate
python script_livre.py
